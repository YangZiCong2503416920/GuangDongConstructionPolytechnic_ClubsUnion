# 变更日志 (Changelog)

所有值得注意的变更都记录在此文件中。

## [v2.1.0] - 2025-11-25

### 修复
- **导航栏玻璃态效果稳定性**：完全移除导航链接的滑动光效，解决悬停时的错位和跳动问题
- **过渡效果优化**：精确控制导航链接的过渡属性，仅对 box-shadow 和 border 应用过渡
- **视觉一致性**：确保玻璃态效果在按钮区域内稳定显示，无位置偏移
- **收藏功能状态更新**：修复收藏按钮点击后状态不立即更新的问题，现在点击后立即显示正确状态
- **收藏按钮参数传递**：修复收藏功能中按钮元素传递问题，确保状态更新准确
- **banner图片显示**：修复banner图片被裁剪问题，使用 `object-fit: contain` 确保完整显示
- **navbar背景色**：调整导航栏背景色，使用更深的深蓝色调，提高对比度和视觉效果

### 改进
- **性能提升**：减少不必要的 CSS 动画计算，提高渲染效率
- **用户体验**：悬停效果更加平滑自然，视觉反馈更清晰
- **收藏功能体验**：收藏按钮点击后立即更新状态，无需刷新页面
- **图片显示效果**：banner图片完整显示，不裁剪内容
- **导航栏视觉效果**：更深的背景色，更好的视觉层次

### 移除
- **滑动光效**：移除 `.nav-link::after` 伪元素及其相关样式，彻底解决滑动效果导致的视觉不稳定问题
