# 广东建设职业技术学院社团融合门户

这是一个为广东建设职业技术学院开发的社团融合门户网站，包含首页、精选活动、所有社团、关于我们四个主要页面。

## 功能特性

- 响应式设计，支持多设备访问
- 社团分类筛选功能
- 社团名称搜索功能
- 社团详情查看和报名功能
- 清晰的导航结构
- 数据分离架构
- 现代化视觉设计
- 收藏功能
- 预留扩展接口

## 页面结构

- 首页：包含Banner图和快速入口
- 精选活动：左侧图片，右侧活动列表
- 所有社团：60+社团分类展示，支持筛选和搜索
- 关于我们：学院社团门户介绍

## 技术栈

- HTML5
- CSS3
- JavaScript
- JSON数据存储
- LocalStorage

## 文件结构

```bash
project/
├── index.html
├── activities.html
├── clubs.html
├── about.html
├── styles.css
├── script.js
└── clubs-data.json
```

## 数据架构

- 社团信息存储在 `clubs-data.json` 文件中
- 通过JavaScript动态加载数据
- 支持数据的快速更新和维护

## 设计特色

- 现代化渐变背景设计
- 玻璃态效果和悬停动画
- 统一的视觉风格
- 优化的用户体验

## 功能亮点

- **收藏功能**: 支持收藏感兴趣的社团，收藏状态持久化存储
- **实时搜索**: 支持按名称和描述搜索，包含防抖优化
- **关键词高亮**: 搜索结果中的关键词会被高亮显示
- **操作反馈**: 提供操作通知和动画反馈
