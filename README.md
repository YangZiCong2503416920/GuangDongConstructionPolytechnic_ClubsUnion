# 广东建设职业技术学院社团融合门户

这是一个为广东建设职业技术学院开发的社团融合门户网站，包含首页、精选活动、所有社团、关于我们四个主要页面。

## 功能特性

- 响应式设计，支持多设备访问
- 社团分类筛选功能
- 社团名称搜索功能
- 社团详情查看和报名功能
- 清晰的导航结构
- 预留扩展接口

## 页面结构

- 首页：包含Banner图和快速入口
- 精选活动：左侧图片，右侧活动列表
- 所有社团：60+社团分类展示，支持筛选和搜索
- 关于我们：学院社团门户介绍

## 技术栈

- HTML5
- CSS3
- JavaScript
